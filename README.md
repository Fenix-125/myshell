## Prerequisites

 - **C++ compiler** - needs to support **C++17** standard
 - **CMake** 3.15+
 
The rest prerequisites (such as development libraries) can be found in the [packages file](./apt_packages.txt) in the form of the apt package manager package names.

## Installing
```
bash compile.sh
```
## Usage
```
./myshell
```
## Internal commands:
- mcd
- mpwd
- mexit
- merrno
- mecho
- mexport
